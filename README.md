## Description
<b>Stack:</b>   
- [Nest](https://github.com/nestjs/nest).  
- [MongoDB Atlas](https://www.mongodb.com/).  
- [Heroku](https://www.heroku.com/).  
  
## Current Feature  
- Get Product    
- Get Restaurant     
- Create Product      
- Create Restaurant   
- Update Restaurant   
- Update Product        
- Delete Product   
- Delter Restaurant  
  
## Upcoming Feature  
- Auth Module  
- JWT authorization for CUD (Create, Update, Delete) access  
- Profile Module

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
